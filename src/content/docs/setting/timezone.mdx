# Setting Timezone

Cek setting
```js
timedatectl
```

:::caution[view]
```               Local time: Thu 2023-09-07 11:45:30 UTC
           Universal time: Thu 2023-09-07 11:45:30 UTC
                 RTC time: n/a
                Time zone: Etc/UTC (UTC, +0000)
System clock synchronized: yes
              NTP service: inactive
          RTC in local TZ: no
```
:::

cek list Timezone

```js
timedatectl list-timezones
```

Set Timezone

```
sudo timedatectl set-timezone Asia/Jakarta
```


Cek setting
```
timedatectl
```
:::caution[view]

```               Local time: Thu 2023-09-07 18:46:45 WIB
           Universal time: Thu 2023-09-07 11:46:45 UTC
                 RTC time: n/a
                Time zone: Asia/Jakarta (WIB, +0700)
System clock synchronized: yes
              NTP service: inactive
          RTC in local TZ: no

```
:::

Identify the timezone you want to configure, and create a symlink :

```
ln -sf /usr/share/zoneinfo/Asia/Jakarta /etc/localtime
```
